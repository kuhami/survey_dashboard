# 数据调研

## Build Setup

```bash
# clone the project
git clone https://github.com/kuhami/survey_dashboard.git

# enter the project directory
cd survey_dashboard

# install dependency
npm install

# develop
npm run dev
```

This will automatically open http://localhost:9528
